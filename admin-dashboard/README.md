# BCC Events Admin Dashboard

Vue 3 + Nuxt 3 admin dashboard for church administrators to manage events, users, and settings.

## Technology Stack

- Vue 3
- Nuxt 3
- PrimeVue
- @bcc-code/design-tokens
- TypeScript
- Service Workers (for offline capability)

## Features

- Event management
- User management
- Payment provider configuration
- Settings and branding
- Reports and analytics
- Live updates via WebSocket

## Getting Started

```bash
npm install
npm run dev
```

## Environment Variables

See `.env.example` for required environment variables.

## Project Structure

```
admin-dashboard/
├── components/     # Vue components
├── pages/         # Nuxt pages
├── composables/   # Vue composables
├── stores/        # Pinia stores
├── utils/         # Utility functions
```
